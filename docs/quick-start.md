# Quick Start (End-to-End)

1) Install deps: `npm install`  
2) Build: `npm run build`  
3) Launch dev extension: `code --extensionDevelopmentPath packages/extension`  
4) In VS Code, select text and run `SocialPostAI: Draft Post`  
5) Choose tone and platform(s); review preview in the SocialPostAI output channel  
6) Choose “Publish now” or “Schedule later” and provide an ISO timestamp (e.g., `2026-01-20T15:30:00Z`)  
7) Keep VS Code open until scheduled posts dispatch
