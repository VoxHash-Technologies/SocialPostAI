# Getting Started

1. Install prerequisites: Node.js 18+, npm 10+, VS Code/VSCodium ^1.85.
2. Clone the repository and install dependencies:
   - `npm install`
3. Build all packages:
   - `npm run build`
4. Launch VS Code with the extension in dev mode:
   - `code --extensionDevelopmentPath packages/extension`
5. Open Command Palette and run:
   - `SocialPostAI: Draft Post`
   - Provide Threads token + user id when prompted
   - Provide Bluesky handle + app password when prompted
6. Preview, publish now, or schedule for later.
