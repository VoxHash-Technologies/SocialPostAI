# Installation

## Requirements
- Node.js 18+ and npm 10+
- VS Code or VSCodium ^1.85

## Steps
1. Install dependencies: `npm install`
2. Build packages: `npm run build`
3. Start extension development host:
   - `code --extensionDevelopmentPath packages/extension`
4. Verify commands appear in the Command Palette (`SocialPostAI: ...`)

## Optional
- Run tests: `npm test`
- Format code: `npm run format`
